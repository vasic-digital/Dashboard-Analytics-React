# @vasic-digital/dashboard-analytics

React dashboard and analytics components for Catalogizer. Provides stats cards, entity distribution visualization, and activity feed.

## Install

```bash
npm install @vasic-digital/dashboard-analytics @vasic-digital/media-types
```

## Components

- `StatsCard` — single metric with label, value, unit, trend indicator
- `EntityStatsGrid` — grid of StatsCards from EntityStats data
- `MediaDistributionBar` — horizontal bar chart for type distribution
- `ActivityFeed` — scrollable list of recent activity events
