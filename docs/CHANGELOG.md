# Changelog — @vasic-digital/dashboard-analytics

## [0.1.0] — 2026-02-21

### Added
- `StatsCard`: metric card with label, value, optional unit and trend arrow
- `EntityStatsGrid`: grid of StatsCards from EntityStats with auto-formatted size
- `MediaDistributionBar`: horizontal proportional bar chart with legend
- `ActivityFeed`: time-ordered activity list with configurable max items
